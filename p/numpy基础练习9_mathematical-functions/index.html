Miscellaneous

Q27. If an element of x is smaller than 3, replace it with 3.

And if an element of x is bigger than 7, replace it with 7.

x = np.arange(10)

out1 = np.clip(x, 3, 7)
out2 = np.copy(x)
out2[out2 < 3] = 3
out2[out2 > 7] = 7

assert np.array_equiv(out1, out2)
print out1

[3 3 3 3 4 5 6 7 7 7]
