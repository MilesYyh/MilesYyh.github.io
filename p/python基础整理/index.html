3 Python控制流

在Python中通常的情况下程序的执行是从上往下执行的，而某些时候我们为了改变程序的执行顺序，使用控制流语句控制程序执行方式。Python中有三种控制流类型：顺序结构、分支结构、循环结构。

另外，Python可以使用分号”;“分隔语句，但一般是使用换行来分隔；语句块不用大括号”{}"，而使用缩进（可以使用四个空格）来表示

3.1 顺序结构

s = '7'
num = int(s) # 一般不使用这种分隔方式
num -= 1 # num = num - 1 
num *= 6 # num = num * 6
print (num)

36
