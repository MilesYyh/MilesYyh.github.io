计算离群值的方法

注意: 均值(average)和标准差(Standard Deviation)只对高斯分布(gaussian distribution)有意义。

#方法1

#直接算
#不过这个是对整个数据而言的，比如你的std就是整个数据而言的
newdf = df.copy()

newdf["x-Mean"] = abs(newdf["Revenue"]-newdf["Revenue"].mean())
newdf["1.96*std"] = 1.96*newdf["Revenue"].std()
newdf["Outlier"] = abs(newdf["Revenue"] - newdf["Revenue"].mean()) > 1.96*newdf["Revenue"].std()
newdf

result: 

#方法2
#分组的方法
newdf = df.copy()
#按照state分组
States = newdf.groupby("State")
#体会下分类的好处！！！
#这样就可以实现，每个State中的类只减去自己对应类的均值了
#就是说它会将第一个类中的全部数据去实现transform(这时候只需要考虑这一类中含有的数据),  再去第二个类。。。。
newdf["Outlier"] = States.transform(lambda x:abs(x-x.mean()) > 1.96*x.std() )
newdf["x-Meaan"] = States.transform(lambda x:abs(x-x.mean()))
newdf["1.96*std"] = States.transform(lambda x:1.96*x.std())
newdf

result: 

#方法3 
#将分组封装为一个函数，再apply用

newdf = df.copy()

States = df.groupby("State")
#和上面的道理一样
def s(group):
    group["x-mean"] = abs(group["Revenue"]- group["Revenue"].mean())
    group["1.96*std"] = 1.96*group["Revenue"].std()
    group["Outlier"] = abs(group["Revenue"] - group["Revenue"].mean()) > 1.96*group["Revenue"].std()
    
    return group

Newdf2 = States.apply(s)
Newdf2

#方法3
#按多个条件分组
#其实就是groupby中又分组了

newdf = df.copy()
#再次强调下这个groupby的大作用，是将数据分组，一组又一组作为一个对象去运行函数之类的
States_month = newdf.groupby(["State",lambda x:x.month])#就又分了月份，即把每年相同月份的数据放在一起了
